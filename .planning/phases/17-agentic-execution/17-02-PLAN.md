    **1. Update `src/ergos/llm/generator.py`** — add method to LLMGenerator:
    ```python
    def create_chat_completion_sync(
        self,
        messages: list[dict],
        tools: list[dict] | None = None,
        max_tokens: int = 512,
    ) -> dict:
        """Call create_chat_completion with model_lock held. Blocking — run from executor."""
        model = self._ensure_model()
        with self._model_lock:
            return model.create_chat_completion(
                messages=messages,
                tools=tools or [],
                tool_choice="auto" if tools else "none",
                max_tokens=max_tokens,
                temperature=0.2,
                stop=["<|im_end|>", "<|endoftext|>"],
            )
    ```

    **2. Create `src/ergos/llm/tool_processor.py`** with class `ToolCallProcessor`:

    Constructor:
    - `generator: LLMGenerator` — for create_chat_completion_sync
    - `registry: ToolRegistry` — for get_tools() and get_impl_map()
    - `executor: ToolExecutor` — for execute(name, args)
    - `system_prompt: str` — same system prompt as LLMProcessor
    - `max_steps: int = 8` — agentic loop cap

    Key method `async def process(self, user_text: str, speak: Callable, llm_processor: LLMProcessor) -> str`:
    - Build messages list: [system_message, ...history_from_llm_processor, user_message_with_/no_think]
    - Get tools list from registry.get_tools()
    - Run agentic loop in asyncio executor (create_chat_completion_sync is blocking):
      - For each step up to max_steps:
        - Call create_chat_completion_sync via loop.run_in_executor
        - If finish_reason != "tool_calls": extract text, break
        - If finish_reason == "tool_calls":
          - Append assistant message (with tool_calls) to local messages
          - For each tool_call: run narration AND tool execution CONCURRENTLY using asyncio.gather:
            ```python
            narration = self._narration_before(tc["function"]["name"])
            args = json.loads(tc["function"]["arguments"])
            # Concurrent: speak narration while tool executes (RESEARCH.md Pitfall 6)
            _, result_str = await asyncio.gather(
                speak(narration),
                self.executor.execute(tc["function"]["name"], args),
            )
            # After both complete, speak completion
            await speak("Done.")
            ```
          - Append tool result messages (role=tool, tool_call_id) to local messages
          - Continue loop
    - After loop: add ONLY the original user message and final response to llm_processor._history (as Message objects)
    - Return final response text

    Narration helper `_narration_before(tool_name: str) -> str`:
    - file_read -> "Let me read that file."
    - shell_run -> "Let me run that command."
    - file_list -> "Let me check what files are there."
    - default -> "Let me check that..."

    Property `has_tools` -> bool from registry.has_tools.
