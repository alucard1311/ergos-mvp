Create ConnectionManager class to manage RTCPeerConnection lifecycle.

1. ConnectionManager class:
   - _connections: set[RTCPeerConnection] = set()
   - _data_channels: set = set()

2. async def create_connection(self) -> RTCPeerConnection:
   - Create RTCPeerConnection()
   - Add to _connections set
   - Register connectionstatechange handler:
     - On "failed" or "closed": clean up and remove from set
   - Return the connection

3. def track_data_channel(self, channel) -> None:
   - Add channel to _data_channels
   - Register close handler to remove from set

4. def get_open_channels(self) -> list:
   - Return channels where readyState == "open"

5. async def broadcast_message(self, message: str) -> None:
   - Send message to all open data channels
   - Use try/except to handle any channel errors

6. async def close_all(self) -> None:
   - Close all connections (await pc.close() for each)
   - Clear _connections and _data_channels sets

7. @property stats(self) -> dict:
   - Return connection count, channel count

<task type="auto">
  <name>Task 2: Create signaling routes</name>
  <files>src/ergos/transport/signaling.py</files>
  <action>
Create aiohttp signaling routes for WebRTC offer/answer.

1. Factory function: create_signaling_app(manager: ConnectionManager) -> web.Application
   - Creates aiohttp Application
   - Registers /offer route
   - Registers on_shutdown handler for cleanup

2. async def offer(request: web.Request) -> web.Response:
   - Parse JSON body: {"sdp": "...", "type": "offer"}
   - Create RTCSessionDescription from offer
   - Get connection from manager.create_connection()
   - Set remote description: await pc.setRemoteDescription(offer)
   - Create answer: await pc.createAnswer()
   - Set local description: await pc.setLocalDescription(answer)
   - Return JSON: {"sdp": pc.localDescription.sdp, "type": pc.localDescription.type}

3. Error handling:
   - Return 400 for invalid JSON or missing fields
   - Return 500 for connection errors
   - Log errors appropriately

4. on_shutdown handler:
   - Call manager.close_all() on app shutdown

Import from aiohttp: web
Import from aiortc: RTCSessionDescription
Import from .connection import ConnectionManager
  </action>
