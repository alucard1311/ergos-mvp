  <action>
Create DataChannelHandler for processing data channel messages.

1. DataChannelHandler class:
   - __init__(vad_processor: VADProcessor, state_machine: ConversationStateMachine)
   - _channels: set for tracking open channels

2. def register_channel(self, channel) -> None:
   - Add channel to _channels set
   - Register message handler: @channel.on("message")
   - Register close handler to remove from set
   - Log channel registration

3. async def handle_message(self, message: str) -> None:
   - Parse JSON from message string
   - Route based on "type" field:
     - "vad_event": call _handle_vad_event
     - "barge_in": call _handle_barge_in
     - Unknown: log warning

4. async def _handle_vad_event(self, data: dict) -> None:
   - Extract event type from data["event"]
   - Call vad_processor.process_raw_event(event_type, data)

5. async def _handle_barge_in(self, data: dict) -> None:
   - Call state_machine.barge_in()
   - Log barge-in event

6. async def broadcast_state_change(self, event: StateChangeEvent) -> None:
   - Serialize event using event.to_dict()
   - JSON dumps the dict
   - Send to all channels where readyState == "open"
   - Use try/except for each channel send

7. def get_state_callback(self) -> Callable:
   - Return async callback function for state machine registration
   - The callback calls broadcast_state_change

8. @property stats(self) -> dict:
   - Return channel count, open channel count
