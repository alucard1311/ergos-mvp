<task type="auto">
  <name>Task 2: Create custom AudioStreamTrack for TTS output</name>
  <files>src/ergos/transport/audio_track.py</files>
  <action>
Create TTSAudioTrack that subclasses aiortc.MediaStreamTrack for streaming TTS audio to WebRTC.

Key implementation details from RESEARCH.md:

1. Class definition:
   - kind = "audio" (required for aiortc)
   - __init__(sample_rate: int = 24000) - Kokoro outputs 24kHz
   - _queue: asyncio.Queue for audio samples
   - _timestamp: int = 0 (cumulative sample count for pts)
   - _samples_per_frame: int = int(sample_rate * 0.020) - 20ms frames

2. async def recv(self) -> AudioFrame:
   - CRITICAL: Must never block. Use asyncio.wait_for with AUDIO_PTIME timeout
   - If no audio available, return silence (zeros)
   - Create AudioFrame from numpy array using AudioFrame.from_ndarray()
   - Set frame.pts = self._timestamp
   - Set frame.sample_rate = self._sample_rate
   - Set frame.time_base = fractions.Fraction(1, self._sample_rate)
   - Increment _timestamp by number of samples
   - Samples must be int16, reshaped to (1, samples) for mono

3. def push_audio(self, samples: np.ndarray) -> None:
   - Put samples on queue (non-blocking put_nowait)
   - Convert float32 TTS output to int16 if needed

4. def clear(self) -> None:
   - Clear queue for barge-in support
   - Use while loop with get_nowait until QueueEmpty

5. Constants:
   - AUDIO_PTIME = 0.020 (20ms frame duration - aiortc standard)

Import from aiortc: MediaStreamTrack, MediaStreamError
Import from av: AudioFrame
Import fractions, asyncio, numpy

AVOID: Blocking calls in recv(). Return silence instead of waiting indefinitely.
