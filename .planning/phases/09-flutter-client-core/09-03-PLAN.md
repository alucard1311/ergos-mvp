       - Factory constructor VADEvent.speechStart() and VADEvent.speechEnd()

    2. Create client/lib/services/vad_service.dart implementing RESEARCH.md patterns:
       - Class VADService with:
         - Private _vadHandler (VadHandler?)
         - Callback onVADEvent (Function(VADEvent)?)
         - Private _speechStartTime (DateTime?) for duration tracking

       - Method `Future<void> initialize()`:
         - Create VadHandler with:
           - mode: VadMode.normal
           - sileroVadModelVersion: SileroVadModelVersion.v5
           - frameSamples: 512 (required for v5 - pitfall #7)
           - sampleRate: 16000
           - positiveSpeechThreshold: 0.5
           - negativeSpeechThreshold: 0.35

         - Set up onSpeechStart listener:
           - Record _speechStartTime
           - Emit VADEvent.speechStart()

         - Set up onSpeechEnd listener:
           - Calculate duration if _speechStartTime set
           - Emit VADEvent.speechEnd() with duration_ms if available
           - Clear _speechStartTime

         - Set up onVADMisfire listener (just ignore false positives)

       - Method `void startListening()`: _vadHandler?.startListening()
       - Method `void stopListening()`: _vadHandler?.stopListening()
       - Method `void dispose()`: _vadHandler?.dispose()

       CRITICAL: Use frameSamples: 512 for v5 model (32ms frames)
