    1. In android/app/build.gradle:
       - Set minSdkVersion to 23 (required for flutter_webrtc)



<task type="auto">
  <name>Task 3: Configure iOS platform</name>
  <files>client/ios/Podfile, client/ios/Runner/Info.plist</files>
  <action>
    1. In ios/Podfile, add post_install hook for WebRTC compatibility:
       ```ruby
       post_install do |installer|
         installer.pods_project.targets.each do |target|
           flutter_additional_ios_build_settings(target)
           target.build_configurations.each do |config|
             config.build_settings['ONLY_ACTIVE_ARCH'] = 'YES'
           end
         end
       end
       ```
    2. In ios/Runner/Info.plist, add inside the main <dict>:
       ```xml
       <key>NSMicrophoneUsageDescription</key>
       <string>Ergos needs microphone access for voice interaction</string>
       ```
    3. Run `cd client/ios && pod install` to install iOS pods
  </action>
  <verify>cd client/ios && pod install succeeds</verify>
  <done>iOS configured with permissions and WebRTC compatibility</done>
</task>
