**1. src/ergos/persona/builder.py (NEW)**
Create TARSPromptBuilder class with section-based sarcasm blending per RESEARCH.md Pattern 1:
- Define TARS_NEUTRAL_SECTIONS dict with keys: identity, style, emotion, humor (humor is empty string at 0%)
- Define TARS_MAX_SARCASM_SECTIONS dict with all four sections populated (deadpan, emotion hints guidance, humor-on-every-turn)
- `build(self, name: str, sarcasm_level: int, memories: list[str], time_context: str) -> str`:
  - sarcasm_level <= 20: use TARS_NEUTRAL_SECTIONS
  - sarcasm_level >= 80: use TARS_MAX_SARCASM_SECTIONS
  - 21-79: use max identity/style, modulate humor frequency ("some" < 50, "most" >= 50)
  - Format name into identity section via .format(name=name)
  - Append time_context if non-empty
  - Append memory entries block if memories list is non-empty: "What you know about the user:\n- entry1\n- entry2"
  - Join sections with "\n\n"
- Identity section at 0%: "You are {name}, a highly capable AI assistant. You are competent, direct, and precise."
- Identity at 100%: "You are {name}, a highly capable AI assistant with a dry wit... like the TARS robot from Interstellar."
- Style section includes "Keep responses to 1-3 sentences. No markdown." at all levels
- Emotion section at 0%: "Omit emotion hints — this is serious mission mode."
- Emotion at 100%: Guides strategic use of *sighs*, *chuckles*, ellipsis for timing

Also in builder.py:
- `try_sarcasm_command(text: str) -> int | None`: Regex pattern r'\b(?:set|change|make|put)\b.{0,20}\bsarcasm\b.{0,30}\b(\d{1,3})\s*%?' — returns clamped 0-100 int or None
- `get_time_context() -> str`: Returns "It is {period} on a {day}." based on datetime.now() — periods: hour<6 "middle of the night", <12 "morning", <17 "afternoon", <21 "evening", else "late at night"
