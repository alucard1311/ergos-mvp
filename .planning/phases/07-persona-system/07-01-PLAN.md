    2. Create src/ergos/persona/types.py with Persona dataclass:
       ```python
       @dataclass
       class Persona:
           name: str
           description: str  # Short bio/description of the assistant
           personality_traits: list[str]  # e.g., ["friendly", "concise", "helpful"]
           voice: str = "af_sarah"  # TTS voice selection
           speaking_style: str = ""  # Optional style notes like "casual" or "professional"

           @property
           def system_prompt(self) -> str:
               """Generate system prompt from persona attributes."""
               # Build prompt from name, description, traits, and style
               parts = [f"You are {self.name}, {self.description}"]
               if self.personality_traits:
                   traits = ", ".join(self.personality_traits)
                   parts.append(f"Your personality is: {traits}.")
               if self.speaking_style:
                   parts.append(f"Speaking style: {self.speaking_style}.")
               parts.append("Keep responses concise for voice interaction.")
               return " ".join(parts)
       ```

    3. Create src/ergos/persona/loader.py with:
