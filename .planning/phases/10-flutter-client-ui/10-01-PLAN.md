  <files>client/lib/widgets/ergos_orb.dart</files>
  <action>
    Create client/lib/widgets/ergos_orb.dart implementing RESEARCH.md patterns:

    Class ErgosOrb extends StatefulWidget with:
    - Properties: serverState (String), onBargeIn (VoidCallback?)
    - Constructor with required serverState, optional onBargeIn

    Class _ErgosOrbState with SingleTickerProviderStateMixin:
    - Private AnimationController _controller (duration: 1200ms)
    - Private Animation<double> _scaleAnimation (Tween 0.95 to 1.05, CurvedAnimation easeInOut)
    - Private Color _currentColor

    In initState():
    - Create AnimationController with vsync: this
    - Create _scaleAnimation with CurvedAnimation
    - Call _updateForState(widget.serverState)

    Method _updateForState(String state):
    - Set _currentColor = _colorForState(state)
    - Switch on state:
      - 'IDLE': controller.stop(), controller.value = 0.5
      - 'LISTENING': controller.repeat(reverse: true)
      - 'PROCESSING': controller.repeat(reverse: true, period: 600ms)
      - 'SPEAKING': controller.repeat(reverse: true)

    Method _colorForState(String state) returns Color:
    - 'IDLE': Color(0xFF6B7280) grey
    - 'LISTENING': Color(0xFF3B82F6) blue
    - 'PROCESSING': Color(0xFFF59E0B) amber
    - 'SPEAKING': Color(0xFF10B981) green

    In didUpdateWidget():
    - If serverState changed, call _updateForState

    In build():
    - GestureDetector wrapping AnimatedBuilder
    - onTap: widget.serverState == 'SPEAKING' ? widget.onBargeIn : null
    - behavior: HitTestBehavior.opaque
    - AnimatedBuilder with animation: _controller
    - Inside builder: Transform.scale with scale: _scaleAnimation.value
    - Inside Transform: AnimatedContainer (duration 300ms) with CustomPaint
    - CustomPaint: size Size(200, 200), painter: OrbPainter(color: _currentColor, glowIntensity: _controller.value)

    In dispose():
    - _controller.dispose()

    CRITICAL: Use SingleTickerProviderStateMixin for vsync to prevent offscreen resource usage
  </action>
