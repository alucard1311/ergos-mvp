<task type="auto">
  <name>Task 1: Create latency metrics module</name>
  <files>src/ergos/metrics.py</files>
  <action>
    Create `src/ergos/metrics.py`:

    1. Import time, statistics, logging, dataclass

    2. Define `@dataclass class LatencyMetrics`:
       - count: int = 0
       - total_ms: float = 0.0
       - samples: list[float] (last 100 samples for percentile calc)

       Methods:
       - record(latency_ms: float): Add sample, update total, trim to last 100
       - p50() -> float: Return median (0 if no samples)
       - p95() -> float: Return 95th percentile (0 if no samples)
       - mean() -> float: Return average (0 if no samples)
       - to_dict() -> dict: Return all stats as dictionary

    3. Define `class LatencyTracker`:
       Tracks timing for a single interaction cycle.

       Attributes:
       - _speech_end_time: Optional[float] = None
       - _first_audio_time: Optional[float] = None
       - _metrics: LatencyMetrics

       Methods:
       - mark_speech_end(): Record time.monotonic() when user stops speaking
       - mark_first_audio(): Record time when first TTS audio is ready
       - compute_latency() -> Optional[float]: Return ms between speech_end and first_audio
       - reset(): Clear current cycle times
       - log_current(): Log current cycle latency and cumulative stats

    4. Add logging on each interaction:
       - "Latency: {current_ms:.0f}ms | P50: {p50:.0f}ms | P95: {p95:.0f}ms | Mean: {mean:.0f}ms (n={count})"
  </action>
  <verify>python -c "from ergos.metrics import LatencyTracker, LatencyMetrics" succeeds</verify>
  <done>Latency metrics module tracks and computes percentiles</done>
</task>
