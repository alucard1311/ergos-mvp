  <name>Task 1: VRAM Monitor module and GPU memory utilities</name>
  <files>src/ergos/core/__init__.py, src/ergos/core/vram.py, src/ergos/hardware.py, tests/unit/test_vram.py</files>
  <behavior>
    - VRAMSnapshot dataclass stores total_mb, used_mb, free_mb, utilization_pct
    - ModelVRAMProfile dataclass stores model_name, estimated_mb, category (stt/llm/tts/vision)
    - VRAMMonitor.snapshot() returns current GPU VRAM usage as VRAMSnapshot
    - VRAMMonitor.snapshot() returns a zero-usage snapshot when no GPU is available (does not crash)
    - VRAMMonitor.register_model(name, estimated_mb, category) tracks expected model allocations
    - VRAMMonitor.budget_check(headroom_mb=4000) returns (fits: bool, total_estimated_mb, available_mb)
    - VRAMMonitor.report() returns a dict with per-model estimates and current GPU snapshot
    - hardware.py get_vram_usage() returns (used_mb, total_mb) tuple using torch.cuda
  </behavior>
  <action>
    1. Create `src/ergos/core/__init__.py` as empty module init.

    2. Create `src/ergos/core/vram.py` with:
       - `VRAMSnapshot` dataclass: total_mb (float), used_mb (float), free_mb (float), utilization_pct (float)
       - `ModelVRAMProfile` dataclass: model_name (str), estimated_mb (float), category (str: "stt"|"llm"|"tts"|"vision")
       - `VRAMMonitor` class:
         - `__init__(self, vram_budget_mb: float = 16384.0)` — stores budget, empty model registry dict
         - `snapshot(self) -> VRAMSnapshot` — uses `torch.cuda.mem_get_info()` if CUDA available, returns zero snapshot if not. mem_get_info returns (free, total) so used = total - free.
         - `register_model(self, name: str, estimated_mb: float, category: str) -> None` — adds ModelVRAMProfile to internal dict
         - `unregister_model(self, name: str) -> None` — removes from dict
         - `budget_check(self, headroom_mb: float = 4000.0) -> tuple[bool, float, float]` — sums all registered model estimates, compares against (vram_budget_mb - headroom_mb), returns (fits, total_estimated, budget_minus_headroom)
         - `report(self) -> dict` — returns {"models": {name: profile_dict}, "gpu": snapshot_dict, "budget_mb": budget, "total_estimated_mb": sum}
         - `registered_models` property returning dict of registered models

    3. Add to `src/ergos/hardware.py`:
       - `get_vram_usage() -> tuple[float, float]` function — returns (used_mb, total_mb). Uses torch.cuda.mem_get_info() if available, returns (0.0, 0.0) if no GPU.

    4. Create `tests/unit/test_vram.py` with tests for all behaviors listed above. Mock `torch.cuda` for deterministic testing — use `unittest.mock.patch` to mock `torch.cuda.is_available` and `torch.cuda.mem_get_info`. Test both GPU-available and no-GPU paths.
  </action>
