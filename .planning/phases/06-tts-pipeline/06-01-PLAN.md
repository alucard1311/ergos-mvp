       - __init__(self, model_path: str, voices_path: str):
         - Store config but don't load model yet (lazy load like LLMGenerator)
         - model_path: Path to kokoro-v1.0.onnx
         - voices_path: Path to voices-v1.0.bin

       - _ensure_model(self) -> Kokoro:
         - Lazy load model on first use
         - Log model loading (can take time)
         - Cache in self._kokoro
         - Import: from kokoro_onnx import Kokoro

       - synthesize(self, text: str, config: SynthesisConfig = None) -> SynthesisResult:
         - Use self._kokoro.create(text, voice=config.voice, speed=config.speed, lang=config.lang)
         - Returns (samples, sample_rate) tuple
         - Calculate duration_ms from samples length and sample_rate
         - Return SynthesisResult

       - async synthesize_stream(self, text: str, config: SynthesisConfig = None) -> AsyncIterator[tuple[np.ndarray, int]]:
         - Use self._kokoro.create_stream(text, voice=..., speed=..., lang=...)
         - async for samples, sample_rate in stream: yield (samples, sample_rate)
         - This is natively async from kokoro-onnx

       - @property model_loaded(self) -> bool
       - @property sample_rate(self) -> int (return 24000 - Kokoro's output rate)
       - close(self): release model resources

    2. Update src/ergos/tts/__init__.py to export TTSSynthesizer

    IMPORTANT: kokoro-onnx's create_stream() is natively async, unlike llama-cpp-python
